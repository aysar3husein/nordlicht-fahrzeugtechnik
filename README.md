# Nordlicht Fahrzeugtechnik — Business Landing Page

A full business website concept for an independent car repair shop, built with **Next.js 15, TypeScript, and Tailwind CSS**.

> Note: This is a portfolio/concept version. The business name, address, contact details, and testimonials shown are fictional placeholders — the layout and code are based on a real client project.

## Features

- Responsive marketing site with Hero, Services, Gallery, Testimonials, Appointment Request, and Location sections
- Accessible appointment request form with client-side validation
- Cookie-conscious Google Maps embed (loads only after explicit user consent)
- German legal pages (Impressum, Datenschutzerklärung) structured per standard requirements
- Scroll-based reveal animations (Framer Motion)
- Fully typed component architecture (Header, Footer, Cards, Buttons, etc.)

## Tech Stack

- **Framework:** Next.js 15 (App Router, Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animation:** Framer Motion
- **Icons:** Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.
